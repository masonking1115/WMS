# Run WMS on a new computer

Prerequisites (install once): [Git](https://git-scm.com/download/win) and [Node.js LTS](https://nodejs.org).

## First time

```bash
git clone https://github.com/masonking1115/WMS.git
cd WMS
npm install
npm run dev
```

Then open http://localhost:5173/ — stop the server with `Ctrl + C`.

## Getting later updates

```bash
git pull
npm install
npm run dev
```
